# PDF整理ツール v0.1.8

## 安定性の改善

- Windowsで更新直後にsidecarの実行ファイルが一時的に使用中となった場合、最大90秒まで起動を再試行します。
- 待機上限は、更新後に約50.5秒継続した共有違反の実測結果へ安全余裕を加えて設定しました。
- リトライ対象はWindowsの共有違反だけに限定し、その他の起動エラーは従来どおり直ちに通知します。

## 動作確認

- sidecar起動リトライの成功、上限到達、対象外エラーを単体テストで確認します。
- Windowsでv0.1.5からv0.1.8へ更新し、更新直後のPDF取込・分割・命名・出力を確認します。
