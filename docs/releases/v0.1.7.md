# PDF整理ツール v0.1.7

## 安定性の改善

- Windowsで更新直後にsidecarの実行ファイルが一時的に使用中となった場合の待機上限を最大15秒へ拡張しました。
- リトライ対象はWindowsの共有違反だけに限定し、その他の起動エラーは従来どおり直ちに通知します。
- 常駐モードとoneshotモードの両方に同じ保護を適用しています。

## 動作確認

- sidecar起動リトライの成功、上限到達、対象外エラーを単体テストで確認します。
- Windowsで署名済みinstallerを導入し、v0.1.5からv0.1.7へのアプリ内更新と更新直後のPDF出力を確認します。
