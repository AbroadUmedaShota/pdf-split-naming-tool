# PDF整理ツール v0.1.5

## 主な変更

- STEP2プレビューをマウスホイール、Enter、Shift+Enterでスクロールできるようにしました。
- STEP3の日本語入力中に、IMEの変換確定で次の原稿へ移動しないようにしました。
- STEP3の完了表示がOCR本文の下端を覆わないようにしました。
- 補助テキストのコントラストを改善しました。

## 安定性の改善

- 0ページPDFを取込時に明示的なエラーとして扱うようにしました。
- 上書きしないPDF出力を原子的に確定し、異常終了時に不完全な最終ファイルが残る可能性を抑えました。
- PDF処理を対話処理とバルク処理の2レーンに分け、出力や全ページ走査中もプレビュー操作を待たせにくくしました。

## 動作確認

- Windows 11で署名済みinstallerの新規インストールを確認しました。
- PDF取込、分割、命名、日本語パスへの出力、状態復元、出力失敗からの復旧、Windowsのファイル・フォルダ選択を確認しました。
- installer、署名、更新manifestのバージョンと配布URLの整合を確認しました。
