# PDF Split Naming Tool Recovery

This is the maintainable recovery implementation for `PDF分割命名ツール_v3_1_1`.

## Run

```powershell
cd recovery
python -m pdf_splitter_tool
```

## Test

```powershell
cd recovery
python -m pytest
```
